## Summary
### HTML
* HTML-документ состоит из тегов
* Полностью тег состоит из открывающего его тега и закрывающего: `<div></div>`
* Контент находится между открывающим и закрывающим тегом
* Некоторые теги не имеют закрывающего тега (`meta`, `input`...)
* Болванка `html` несколько усложнилась:
```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
    </head>
    <body></body>
</html>
```
* Указывааем `doctype` и `charset`
* Заголовок окна (вкладка браузера) указывается с помощью тега `title`:
```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Заголовок во вкладке</title>
    </head>
    <body></body>
</html>
```

### DOM
* DOM - document object model - объектная модель документа
* DOM - набор взаимосвязанных объектов (в виде дерева), которые связаны с соответствующими тегами в разметке `html`
* Объект элемента - это способ получать, изменять и добавлять какие-то свойства конкретному элементу в разметке
